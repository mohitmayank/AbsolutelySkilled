# social-media-scheduling

social-media-scheduling is a production-ready AI agent skill for claude-code, gemini-cli, openai-codex, and mcp. It plans and packages full periods of social content so posts can be reviewed, approved, and scheduled.

```bash
npx skills add AbsolutelySkilled/AbsolutelySkilled --skill social-media-scheduling
```

```bash
npx skills add agentreacher/skills --all
```

## Overview

Use this skill to turn a rough campaign goal into a scheduling-ready social media plan. It covers content pillars, cadence planning, platform-native adaptation, calendar building, approval status, missing assets, and review notes.

It is useful for launches, founder-led marketing, SaaS campaigns, AI tooling, creator businesses, and agency social workflows.
